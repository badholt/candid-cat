### Installation

1. Running the Meteor JS server requires a `settings.json` file with SMTP credentials

>

    meteor --settings private/settings.json

>*settings.json*
>

    {"MAIL_URL": "smtp://smtp://postmaster@domain.com:password@domain.com:PORT"}
